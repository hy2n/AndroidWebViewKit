# 안드로이드 WebView 앱 킷

이것은 WebView를 활용하여 기본 앱을 만들고 JavaScript와 앱 간 통신을 가능케 하는 간단한 안드로이드 앱 킷입니다.

## 기능

- 안드로이드 앱 내에서 WebView에서 웹 페이지를 로드합니다.
- JavaScript가 안드로이드 앱과 통신할 수 있습니다.
- JavaScript에서 알림을 안드로이드 앱으로 전송합니다.
